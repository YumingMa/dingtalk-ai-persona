# 钉钉 AI 分身 · 用户使用指南

> 一次授权，之后直接说话，AI 以你的身份帮你操作钉钉。

---

## 第一步：首次授权（只做一次）

### 1. 安装 dws

**macOS / Linux：**
```bash
curl -fsSL https://raw.githubusercontent.com/DingTalk-Real-AI/dingtalk-workspace-cli/main/scripts/install.sh | sh
```

**Windows（PowerShell）：**
```powershell
irm https://raw.githubusercontent.com/DingTalk-Real-AI/dingtalk-workspace-cli/main/scripts/install.ps1 | iex
```

### 2. 登录钉钉

> 如果是 SSH 远程服务器，必须加 `--device`

```bash
dws auth login --device
```

按提示在手机上扫码完成企业授权。

### 3. 导出 token

```bash
dws auth export --base64
```

复制输出的一长串字符（以 `H4sI` 开头）。

### 4. 绑定到 AI 分身

在钉钉里找到「AI 分身」机器人，发送：

```
/bind H4sIAAAA...（你复制的内容）
```

看到「✅ 授权绑定成功」即完成。**之后不需要再做这些步骤。**

## 第二步：日常使用

绑定完成后，直接在钉钉里对机器人说你想做什么，**不需要记任何命令**。

### 📄 文档操作

```
帮我读一下这个文档：
https://alidocs.dingtalk.com/i/nodes/xxxxx

搜索一下关于「分布式数据库」的文档

帮我在知识库里新建一篇文档，标题是「XX方案」，内容是...
```

### 💬 发消息

```
帮我告诉张三：明天的会议推迟到下午三点

帮我在研发部群里发一条消息：本周五下午有技术分享，请大家准时参加

把这份总结发给邵娜
```

### 📊 日志周报

```
帮我看看本周团队都提交了哪些周报，做个总结

读一下刘东南的周报

分析一下本周大家的工作进展
```

### ✅ 待办事项

```
查一下我有哪些未完成的待办

帮我新建一个待办：下周一前完成接口文档
```

### 📅 日程安排

```
查一下我今天有什么日程

明天下午我有空吗？
```

### 🔍 搜索

```
帮我找一下李永帅的 userId

搜一下「港美断网测试」相关的文档
```

## 进阶用法

### 多轮对话

AI 分身记得你们的对话历史，可以追问：

```
用户：帮我读一下这篇文档：[URL]
AI：（读取并总结内容）

用户：把这个总结发到研发部群
AI：（直接发送，不需要再粘贴内容）
```

### 清除记忆

```
/clear
```

重新开始一段新对话。

### 查看帮助

```
/help
```

## 注意事项

1. **权限边界**：AI 只能读取和操作你自己有权限的内容，无法访问他人的私密文档
2. **发消息前确认**：发消息给他人时，AI 会先告知你将要发送的内容，确认后再发
3. **token 过期**：如果 AI 回复「授权失效」，重新执行第一步的 2-4 步骤重新绑定即可
4. **隐私安全**：所有操作均以你本人身份执行，等同于你亲自操作钉钉

## 常见问题

**绑定后 AI 说「授权失效」？**
钉钉 token 有效期约 30 天。重新执行 `dws auth login --device` 和 `dws auth export --base64`，再发 `/bind` 重新绑定即可。

**发送的消息能撤回吗？**
可以，告诉 AI「撤回刚才发给 XXX 的消息」。

**AI 能替我审批 OA 吗？**
暂不支持自动审批，但可以帮你查看待审批列表。

**在群里也能用吗？**
可以，在任何有 AI 分身机器人的群里 @ 它即可，机器人会以你的身份处理请求。
